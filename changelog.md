# Changelog

All notable changes to this project will be documented in this file. This format follows a simplified version of [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1] – 2025‑10‑17

### Added

- Initial release of the OverKill Hill P³ Abrahamic Reference Engine assets.
- Generated 15 knowledge articles covering Judaism, multiple Christian traditions, Islam, Hinduism, Buddhism, cross‑comparative methods, translation licensing, glossaries, and reserved placeholders. Each article includes a YAML header and extended body text to meet token requirements.
- Added `translation_matrix.md` mapping Bible edition codes and Qur’an translation IDs to friendly names and notes.
- Added `settings_schema.json` defining configurable user preferences: default tradition, Bible version, Qur’an translation IDs, Sefaria language, and commentary level.
- Added `test_queries.md` with 30 example prompts covering retrieval, comparison, ambiguity, settings, and refusal scenarios.
- Added `ui_copy.md` containing on‑boarding instructions, usage tips, and error messages for a neutral, scholarly interface.
- Added `CHANGELOG.md` seed entry describing initial release.
- A future `README.md` will provide installation steps for the GPT builder (see README in this project).
